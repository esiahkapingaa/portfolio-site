Phylogenetic Analysis of Tinospora Species

A Molecular Approach to Genetic Relatedness and Taxonomic Resolution

Overview

This project explores the phylogenetic relationships among various Tinospora species using DNA sequence data sourced from GenBank. Tinospora, a genus of medicinally significant climbing shrubs in the Menispermaceae family, exhibits wide morphological diversity and ambiguous classification across geographic ranges. Our goal was to clarify genetic divergence and relatedness using molecular phylogenetics.

Objectives

  • Identify suitable molecular markers (genes) for analyzing Tinospora spp.
  • Retrieve and curate nucleotide sequences from GenBank.
  • Align and clean sequences using computational tools.
  • Construct phylogenetic trees to infer evolutionary relationships.
  • Interpret clades and divergence in context of morphology and taxonomy.

Methods & Tools

  1. Sequence Retrieval: Downloaded gene sequences from GenBank; verified using BLASTn.
  2. Multiple Sequence Alignment: Used ClustalW in MEGA X to align and clean data.
  3. Model Selection: Best-fit substitution model determined via model test (e.g., Tamura-Nei).
  4. Tree Construction: Built ML and NJ trees with 1000 bootstrap replicates.
  5. Annotation: Compared with UniProtKB for functional insight and divergence context.

Hypothetical Results

  • T. cordifolia and T. sinensis grouped with 98% bootstrap support.
  • T. crispa appeared basal, indicating early divergence.
  • matK region showed more variability than rbcL.
  • Possible cryptic species suggested by divergence patterns.

Tools Used

MEGA X: Tree building & alignment
NCBI GenBank: Sequence data
BLASTn: Identity confirmation
UniProtKB: Functional annotation
ClustalW: MSA processing

Applications

  • Supports drug discovery from species-specific metabolites
  • Clarifies Tinospora taxonomy and evolution
  • Guides conservation and correct ID for pharmacognosy

All visualizations and maps are interactive. Data is updated regularly. Ethical considerations were applied regarding reporting accuracy and transparency.
