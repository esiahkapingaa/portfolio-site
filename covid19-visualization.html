Global COVID-19 Trends: Interactive Visualization Using WHO Data

This project leverages real-time data from the World Health Organization (WHO) to visualize global COVID-19 trends, including infection rates, recovery trajectories, and geographical impact. Using Python libraries like Plotly, Folium, and Pandas, the project builds an interactive dashboard and maps to make epidemiological trends visually accessible.

Objectives

  • Connect to the WHO API to retrieve up-to-date COVID-19 data (cases, deaths, recoveries)
  • Build an interactive world map showing infection hotspots and recovery distribution
  • Plot daily trends for confirmed cases, deaths, and recoveries by country/region
  • Enable user filtering by date range and country
  • Extract insights into pandemic progression and regional differences

Data Sources

Source Description
WHO API Official COVID-19 data
Country GeoJSON Country shapes for Folium maps
World Bank (optional) Demographic overlays (e.g., population)

WHO COVID-19 Data Endpoint:
https://covid19.who.int/page-data/table/page-data.json

(Note: uses indirect GraphQL structure – alternatives include CSSEGISandData or COVID19 API)

Methods & Tools

  1. Data Retrieval: Accessed WHO or COVID19 API, parsed JSON into Pandas, cleaned missing/inconsistent values.
  2. Data Wrangling: Aggregated case data, calculated rates, merged with GeoJSON country coordinates.
  3. Mapping (Folium): Built choropleth maps and circle marker layers with popups.
  4. Visualization (Plotly): Created interactive time series plots, bar charts, and optional animations.

Example Insights (Hypothetical)

  • India, USA, and Brazil exhibited consistent surges aligned with policy delays.
  • Recovery rates rose significantly post-July 2021 with mass vaccination campaigns.
  • African nations showed underreporting patterns — visualized as data sparsity.
  • Reinfection waves (Delta, Omicron) visualized in animated strain-period plots.
